# 🏠 House Price Prediction App

This project is a Machine Learning-based web application that predicts house prices based on user inputs.

## 🚀 Features
- Real-time prediction using Streamlit
- Interactive UI with sliders
- Trained using Ridge Regression (R² ≈ 0.91)

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## 📊 Model Performance
- R² Score: 0.91
- MAE: ~80,000

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
